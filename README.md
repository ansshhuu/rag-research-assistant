# RAG Research Assistant

An end-to-end Agentic RAG system built in modular sub-projects.

## Project Structure

| Sub-project | Description | Status |
|---|---|
| p1-ingestion | Document ingestion pipeline |
| p2-query-engine | RAG query engine + FastAPI |
| p3-agent | Agentic reasoning layer | 
| p4-docker | Dockerization |
| p5-cicd | CI/CD pipeline | 
| p6-frontend | Frontend + observability |

## Tech Stack
- Python 3.11+
- sentence-transformers, ChromaDB, LangChain
- FastAPI, LangGraph, Docker, GitHub Actions

## How to run
See each sub-project's own README.md for setup instructions.
